Chat Room
=========

Terminal-style chat room created by Cursor.

![Demo](img/demo.png)

Live Demo
---------

[http://8.138.36.150/chatroom](http://8.138.36.150/chatroom)

Features
--------

- [x] Random ONEPIECE Characters.
- [x] Multi-line messages.
- [x] Copy messages.
- [x] Resize chat area.

Running
-------

```
go run main.go
```

Open `index.html` in browser.
